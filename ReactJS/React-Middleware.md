## 미들웨어(Middleware)
```
리덕스를 사용 하면서 비동기 작업을 다룰 때는 미들웨어가 있어야 더욱 손쉽게 상태를 관리 된다.

미들웨어는,
액션이 디스패지 되어서 리듀서에서 이를 처리하기전에 사전에 지정된 작업을 성절하는 것.
액션과 리듀소 사이의 중간자.

리듀서가 액션을 처리하기전에 미들웨어가 할 수 있는 작업
ex)
1. 콘솔 로그 기록.
2. 액션 취소
3. 다른 종류들의 액션들을 추가적으로 디스패치 할 수 있다.
```

```
const loggerMiddleware = state => next => action => {
  /* 미들웨어 내용 */
}
