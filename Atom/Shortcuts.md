# Atom 단축키

>### General
>`cmd + ,` : 환경 설정
>`shift + cmd + p` : 명령 command
>`cmd + p` : 파일 열기
>`cmd + b` : 찾아보기 한 파일 열기
>`cmd + 숫자` : 탭 이동
>`ctrl + shift + l` : 문법 선택
>`ctrl + shift + m` : 마크다운 미리 보기
>`ctrl + .` : 키 바인딩 확인
>`cmd + k b` : LNB 보기 및 숨기기
>`alt + cmd +i` : 개발자 도구

>### Editing
